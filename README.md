# ONU - SDG

## Installation

### Step 0 - Environment needed:
- nvm - https://github.com/creationix/nvm
- node v10 via nvm - ```nvm install 10```

### Installation
- Clone this repo
- set node 10 version ```nvm use 10```
- run ```npm install```
- run ```gatsby develop``` or ```npm start```

### Build
- run ```gatsby build``` or ```npm buld```
